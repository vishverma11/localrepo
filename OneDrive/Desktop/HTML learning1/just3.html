3.2 Introduction to CSS

CSS is an abbreviation for Cascading Style Sheets. CSS works with HTML and other Markup Languages
(such as XHTML and XML)to control the way the content is presented. Cascading Style Sheets is a means to
separate the appearance of a webpage from the content of the webpage. CSS is the recommendation of the
World Wide Web Consortium(theW3C)

The concept of style sheet is very simple. First, the designer creats a standard Web page, using standard
HTML tags. It is design so that it can be displayed properly in the browser that do not support style sheets.
Now, suppose designer wants the heading(H1) should be display in red colorand the paragraph (P)
should be display in blueand Helvectica font.To accompish the, the author creats style rules that format
the content on the web page in the manner that he or she chooses.

Style sheets would thus combine HTML tags (H1) with properties(such as color:red) to format each HTML
tag. In the case of style sheets, an HTML tag is used as a selector. The property and value of the selector are
combined into what is called declaration. Following are some example of style rules:

H1 {colour: RED}
P {FONT-FAMILY: HELVECTICA; COLOUR:BLUE}

In the above stylesheet tag H1 and P is used as a selector and their properties such as colour and font-family is
used as a declaration part.
